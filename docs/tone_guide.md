# SOARA V2 - Guide de Ton d'Ecriture

Version: 2.0
Statut: Style guide production

## 1. Intention
Le texte Soara doit etre:
- clair
- martial
- concret
- sans emphase fantasy excessive

## 2. Regles de style
- Phrases courtes a moyennes.
- Verbes d'action precis.
- Peu d'adjectifs decoratifs.
- Eviter le mystique flou.

## 3. Champ lexical privilegie
- rythme, intention, engagement, sequence, lecture, pression, consequence.

## 4. Champ lexical a limiter
- epique, legendaire, cosmique, divin, cataclysmique.

## 5. UI narrative
- Une information = une ligne utile.
- Toujours lier action -> consequence.
- Expliquer le "pourquoi" avant le "quoi" quand possible.

## 6. Exemples
## Bon
- "Tu valides trop tot. Ton energie chute."
- "Il lit ta garde et casse ton tempo."

## A eviter
- "Une force mystique indefinissable t'envahit."
- "Un destin ancien guide ta lame."

## 7. Notes de coherence
- Aura = discipline, pas sorcellerie.
- Le log est un document de combat, pas un effet visuel.
- Les factions parlent selon leurs valeurs politiques.
